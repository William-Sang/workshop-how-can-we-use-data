## 📝 使用说明

### 对于AI助手 (Cursor/Claude等)
```markdown
当用户要求完成某个任务时：
1. 找到对应的TODO项目
2. 执行相关的代码修改
3. 将状态从 ❌ TODO 改为 ✅ DONE
4. 更新完成时间为当前时间
5. 更新项目概览中的统计数据
6. 在更新日志中添加一条记录
```

### 任务ID规则
- 格式: `TODO-XXX` 或 `DONE-XXX`
- XXX为三位数字，按创建顺序递增
- 完成后前缀改为DONE，但保持原ID号

### 元数据字段说明
- **状态**: ❌ TODO | ✅ DONE | 🔄 IN_PROGRESS | ⏸️ BLOCKED
- **优先级**: P1-高 | P2-中 | P3-低
- **分类**: 功能分类标签
- **时间格式**: YYYY-MM-DD HH:MM:SS
